This Plugin is used to automatically Ping new posts to a variety of online blogs. For use with Pingler.com


=== Description ===

Everytime you Publish a new post, this plugin will automatically 
send the update information to Pingler.com so the whole world can 
be notified that you keep your content fresh.



=== Installation ===

This section describes how to install the plugin and get it working.

1. Upload `pingler.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Update your settings in the Pingler Options Page under the settings menu
	a. Get your API key from your PIngler account and insert it here.
	b. All additional options will appear once you have entered your API key.
4. There is no 4, like we was going to make it that difficult
